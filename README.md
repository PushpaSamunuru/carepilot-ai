# 🚀 CarePilot AI

**AI-Powered Healthcare Operations Assistant**

CarePilot AI is a modern healthcare operations platform that helps clinics and healthcare teams streamline patient intake, generate AI-assisted summaries, manage follow-ups, and improve administrative workflows.

> **Disclaimer:** This project is a portfolio/demo application intended for administrative workflow demonstrations only. It does **not** provide medical diagnosis or treatment recommendations.

---

## ✨ Features

- 🏥 Patient Intake Management
- 🤖 AI-Style Intake Summaries
- 📋 Patient Dashboard
- 📄 PDF Report Generation
- 📧 Follow-Up Email Preview
- 🔍 Search & Filter Patients
- 📝 Staff Notes
- 📊 KPI Dashboard
- 💾 Local Storage Persistence
- 📱 Responsive Design

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/PushpaSamunuru/carepilot-ai.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
